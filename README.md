## ✔️ Descrição
 Cadastro Mania é um sistema onde apliquei o conhecimento adquirido de ReactJs e Nextjs, salvando os dados no firebase('Banco de dados').

## ✔️ Funcionalidades
- Salvar
- Excluir
- Alterar
- ObterTodos

## ✔️ Tecnologias utilizadas
- JavaScript
- NextJs
- Firebase
- Type Script
- Tailwind
